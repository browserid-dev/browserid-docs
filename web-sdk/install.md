# Install the Web SDK

## Prerequisites
1. Sign up for a Zeroness account at [https://dash.zeroness.dev](https://dash.zeroness.dev)

## Installation

:::tabs
== npm
```shell
npm add @zeroness/web
```
== pnpm
```shell
pnpm add @zeroness/web
```
== yarn
```shell
yarn add @zeroness/web
```
== bun
```shell
bun add @zeroness/web
```
:::
